# 今日の元気パズル

60代・70代の方がスマホで短い脳活パズルを遊び、その結果を家族の見守りや会話のきっかけにするための小さなWebアプリ試作品です。

## 大切にしていること

- 本人には「検査」ではなく、楽しい1分の習慣として使ってもらう
- 家族には結果だけでなく、連絡のきっかけになるやさしい情報を届ける
- 個人情報、電話番号、通知先の秘密情報はコードに直接入れない
- 認知症の診断や医療判断は行わない

## いま入っている機能

- スマホ向けの脳活パズル7問
- 色、形、向き、見え方を使った問題
- 正解・不正解のフィードバック
- 結果画面
- 家族への通知文のイメージ

## 今後の予定

- LINE、メール、SMSのいずれかで家族へ通知する
- 回答時間や利用日の変化を記録する
- 本人が使いやすい文字サイズ、ボタンサイズを調整する
- 3Dパズルをより分かりやすくする

## 使い方

`index.html` をブラウザで開くと動きます。

まだ試作品なので、通知送信や個人データ保存は入れていません。
